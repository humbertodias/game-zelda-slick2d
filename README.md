# Zelda Game with Slick2D

RPG game like Zelda using Slick2D Library

## Prerequires

1. Java 8+
2. Curl
## How to Play


```
curl -L https://github.com/humbertodias/game-zelda-slick2d/releases/download/v0.1/zelda-slick2d-0.0.1.jar -o zelda.jar && java -jar zelda.jar
```
